    Interface CallbacksBeta

    Callbacks used to get status both during and after the id verification process.

    Param: onComplete

    Called when all stages in an id verification process has been completed.

    Param: onUserDismissed

    Called when the user aborts a verification process by pressing the cancel button.

    Param: onClientConnectionLost

    Called when the connection to the Web SDK running on the second device is lost.

    Param: onError

    Called when there is an error in the id verification process.

    Param: onStarted

    Called when the id verification process has started.

    Param: onTransactionCreated

    Called when an id verification transaction has been created.

    Param: onDocumentFeedback

    Called when the users document type has been identified.

    Param: onFaceMatchFeedbackWithSource

    Called when the face matching has completed.
